# Minecraft Username UUID Generator

[Live website](https://yureitzk.github.io/minecraft-username-uuid-generator)

## Features

- Supports both online (Mojang via [api.ashcon.app](https://github.com/Electroid/mojang-api)) and offline UUIDs
- Batch process multiple usernames at once
- Drag and drop text files with usernames for easy import
- Export as JSON or plain text format

## Requirements

Running and building requires `npm` and `Node.js`.

## Building

```sh
npm install

npm run build

# If you want to run a debug version
npm run dev
```
